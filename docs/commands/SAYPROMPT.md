# #SAYPROMPT

## <u>SAYP</u>ROMPT

### **Syntax:** #SAYP text

> Display text to the screen

### **Related:** [#SHOW](SHOW.md), [#SHOWPROMPT](SHOWPROMPT.md), [#ECHO](ECHO.md), [#ECHOPROMPT](ECHOPROMPT.md), [#SAY](SAY.md)

### **Examples:**
`#sayp %{random}`
> Will display a random number to the screen
