# #SAY

## <u>SA</u>Y

### **Syntax:** #SA text

> Display text to the screen and append newline

### **Related:** [#SHOW](SHOW.md), [#SHOWPROMPT](SHOWPROMPT.md), [#ECHO](ECHO.md), [#ECHOPROMPT](ECHOPROMPT.md), [#SAYPROMPT](SAYPROMPT.md)

### **Examples:**
`#sa %{random}`
> Will display a random number to the screen
