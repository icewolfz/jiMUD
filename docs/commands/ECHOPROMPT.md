# #ECHOPROMPT

## <u>ECHOP</u>ROMPT

### **Syntax:** #ECHOP text

> Display text to the screen

### **Related:** [#SHOW](SHOW.md), [#SHOWPROMPT](SHOWPROMPT.md), [#ECHO](ECHO.md), [#SAY](ECHO.md), [#SAYPROMPT](SAYPROMPT.md)

### **Examples:**
`#echop %{random}`
> Will display a random number to the screen
