# #ECHO

## <u>EC</u>HO

### **Syntax:** #EC text

> Display text to the screen and append newline

### **Related:** [#SHOW](SHOW.md), [#SHOWPROMPT](SHOWPROMPT.md), [#ECHOPROMPT](ECHOPROMPT.md), [#SAY](SAY.md), [#SAYPROMPT](SAYPROMPT.md)

### **Examples:**
`#ec %{random}`
> Will display a random number to the screen
